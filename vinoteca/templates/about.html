vinoteca is a wine purchase tracker and review system

Adding wines

To add a new purchase click the new purchase button in the top navigation bar. If this wine is already in the system, select "Previously Purchased" and find the wine using the search utility. If this is the first time you've purchased this wine, select "First-time Purchase" and enter the relevant information.

Viewing wines

Go to "View Wines" to sort, filter and view wine information.

Editing wines

In order to edit a wine's information, click on any wine to go its profile page. Then hover over the menu button on the left side and click the pencil icon. Make your changes and then click "Confirm Edits" To edit purchase details, simply click the pencil icon next to the purchase you would like to edit in the wine's "Previous Purchases" table.

Bugs and suggestions

Send me an email with any problems or bugs you encounter and any suggestions you have for improvements.
